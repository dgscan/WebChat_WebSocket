# WebChat_WebSocket
General &amp; Private Chat Room
========


Installation
------------
Install node.js at: https://nodejs.org/en/download/
and install necessary JavaScript libraries:

    $ npm install

After finished packages installed correctly, run:

    $ node server.js

Open index.html
